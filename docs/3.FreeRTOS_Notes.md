Create Task:
```c
xTaskCreatePinnedToCore(
    TaskFunction_t pvTaskCode, //Task Function
    const char *const pcName, //Task Name
    const uint32_t usStackDepth, //Number of variables allocated in the stack the function has: will allocate 32*usStackDepth bits
    void *const pvParameters, //Function arguments
    UBaseType_t uxPriority, //Task priority
    TaskHandle_t *const pvCreatedTask, //Used to get a task handle so changes can be made to the task later(delete for example)
    const BaseType_t xCoreID //Core that will run the task: 0 or 1 in the case of the ESP32
);
```

Delete Task:
```c
vTaskDelete(
    TaskHandle_t xTaskHandle //The task handle of the function we want to delete
);
```
